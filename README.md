# NLPipeline-Agent

An academic-themed NLP (Natural Language Processing) pipeline visualization tool built with **MiMo V2.5**.

## Features

- Text analysis with entity highlighting
- Sentiment analysis with visual meter
- Part-of-speech tagging visualization
- Named entity recognition (NER)
- Keyword extraction
- Text statistics (word count, sentence count, readability score)
- Academic/research-themed UI design

## Tech Stack

- Vanilla HTML5, CSS3, JavaScript (ES6+)
- Rule-based NLP simulation
- CSS sentiment meters and entity highlights
- Zero dependencies

## Getting Started

1. Open `index.html` in a modern web browser
2. Enter or paste text into the analysis panel
3. Click "Analyze" to run the NLP pipeline
4. View results across multiple analysis panels

## Analysis Modules

- **Tokenization**: Word and sentence segmentation
- **POS Tagging**: Parts-of-speech classification
- **NER**: Named entity extraction
- **Sentiment**: Positive/Negative/Neutral scoring
- **Keywords**: TF-based keyword extraction
- **Readability**: Flesch-Kincaid approximation

Built with MiMo V2.5 AI assistance.
